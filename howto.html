User Guide

1. How to use Location Identifier

Start by importing your CSV file using the buttons in the header or the center of the screen. The app will process the data and start showing video clips from the timestamps provided in your file.

  • The video player will automatically start at the correct timestamp.
  • Click Save often to ensure your progress is kept in browser storage.
  • The app will auto-download a backup CSV on save for your records.

2. Bulk Edit

Efficiency is key. The app groups consecutive seconds from the same video automatically. When you change a location in the dropdown, every row in the current group is updated at once.

3. New Location (Add Custom)

If a required location isn't in the list:

  • Select "undefined" from the dropdown.
  • Type the new location name in the input box that appears.
  • Click the (+) button or press Enter.
  • The new location is permanently added to your personal dropdown list.

4. Move to specific location

Jump exactly where you need to go using real row numbers from your CSV:

  • Click on the row number (e.g., "45 / 20000") to type a specific row you want to find.
  • The app will automatically find the group containing that row.
  • Use the Time Stamps badges to jump to specific seconds within a video.
  • The app auto-jumps to the first unlabeled row on startup.

6. CSV Arrangement

To optimize your workflow, you can rearrange the entire dataset chronologically:

  • Click the Arrange CSV button in the top header.
  • The app sorts all data by Event Date, then by Video URL, and finally by Timestamp.
  • This ensures that consecutive seconds of the same play are always grouped together, maximizing efficiency.
  • A "Master" CSV copy is automatically downloaded to preserve this optimized order for future use.
