# Location Identifier

A premium web-based tool for identifying and labeling locations in video broadcasts. Designed for efficiency, this tool supports bulk editing, keyboard shortcuts, and custom location management.

## 🚀 Key Features

### 1. How to use Location Identifier
- **Import CSV**: Start by clicking the **"Import CSV"** or **"Choose CSV File"** button. Select the "Leafs Broadcast Loc Enhancement Breakouts" file.
- **Identification**: The app automatically groups consecutive seconds from the same video.
- **Save & Persist**: Click **"Save"** to store your progress in browser memory and download a backup CSV. The app resumes exactly where you left off on reload.

### 2. Bulk Edit
- The tool automatically identifies continuous seconds. When you select a location from the dropdown, it **updates all rows in the current group** simultaneously.
- Look for the **yellow indicator** in the info panel to see how many rows are being updated at once.

### 3. New Location (Add Custom)
- To add a location not in the list, select **"undefined"** (or press keyboard key **`4`**).
- A text box will appear. Type the new location name and press **Enter** or click the **"+"** button.
- The new location is added to your dropdown and saved for future use.

### 4. Move to specific location
- **Editable Counter**: You can type a specific group number directly into the navigation counter (e.g., `1 / 5458`) and press **Enter** to jump there.
- **Navigation Buttons**: Use the **"Previous"** and **"Next"** buttons to cycle through groups.
- **Keyboard Arrows**: Use **`ArrowLeft` / `A`** for Previous and **`ArrowRight` / `D`** for Next.

### 5. Other / Advanced
- **Keyboard Shortcuts**:
  - **`1`**: Select "Bench Signage"
  - **`2`**: Select "Static Dasherboard"
  - **`3`**: Select "Vomitory"
  - **`4`**: Select "undefined" (Custom)
  - **`Ctrl+S / Cmd+S`**: Save Progress
- **Auto-Jump**: On start, the app automatically finds and jumps to the first row that hasn't been labeled yet.
- **CSV Export**: Use the **"Export CSV"** button in the header at any time to download your current data.
